# C++_Calculator

문자열 입력으로 작동하는 계산기입니다.

기존의 C_Caculator를 C++로 고쳐보았습니다.

1. 문자형태의 숫자를 숫자형태로 저장
2. 덧셈, 뺄셈 처리
3. 곱셈, 나눗셈 처리
4. 괄호 처리
5. 괄호 안의 괄호 처리
6. 실수 숫자 처리 (소수점 입력)
